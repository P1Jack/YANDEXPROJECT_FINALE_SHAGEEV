Название: GUI Менеджер паролей P1QtPass
Описание проекта:

Во многих современных программных системах основная слабость часто заключается в пароле пользователя.
Это особенно очевидно по многочисленным утечкам баз данных различных сервисов, которые показывают, что
люди часто используют слабые пароли. Например, qwerty или password123456.
Или одни и те же пароли в разных сервисах, что приводит к множественной компрометации в случае утечки пароля в одном из сервисов.
Поэтому очень важно, чтобы пользователи выбирали надежные пароли для защиты своих учетных записей, но надежные пароли зачастую
длинные и незапоминющиеся. Еще более проблематичным является то, что у пользователя обычно есть много различных служб, которые используют аутентификацию по паролю, и в результате пользователю приходится вспоминать много разных паролей. Одним из способов решения этой проблемы для пользователей является использование менеджера паролей. 
Менеджеры паролей позволяют пользователям очень удобно использовать уникальный надежный пароль для каждого сервиса требующей аутентификации по паролю. 
Однако, учитывая конфиденциальность данных, содержащихся в менеджере паролей, требуется большая осторожность для безопасного хранения аутентификационных данных.

В этом проекте будет реализован менеджер паролей с удобным графическим интерфейсом и необходимыми функциями для работы с ним. 
Важным аспектом является использование криптостойкого шифрования паролей в базе данных, иначе теряется смысл задумки.



Техническое задание:
0. Реализация криптостойкового шифрования паролей (AES) в базе данных
1. Добавление/редактирование/удаление паролей
2. Поиск по названия сервиса, логину, категориям
3. Экспорт/импорт базы данных, для переноса на другое устройство
4. Автоматическая генерация надежных паролей (по желанию пользователя)
5. Автоматическое копирование пароля в буфер обмена
6. Смена мастер-ключа
7. Удобный графический интерфейс на PyQT
8. Упаковать программу в exe
9*. Кроссплатформенность
